Configurar Supabase

Conecte ao Supabase para sincronizar em tempo real entre todos os participantes.

Suas credenciais são salvas apenas localmente no navegador (LocalStorage) e usadas para acessar a API REST do Supabase diretamente. Nunca passam por servidores intermediários.

Antes de conectar, crie as tabelas no Supabase SQL Editor:

CREATE TABLE IF NOT EXISTS agenda_perfis (
  id TEXT PRIMARY KEY,
  nome TEXT NOT NULL,
  cor TEXT NOT NULL,
  avatar TEXT NOT NULL
);

CREATE TABLE IF NOT EXISTS agenda_disponibilidade (
  perfil_id TEXT NOT NULL,
  semana_ref TEXT NOT NULL,
  dia_semana INT NOT NULL,
  slot_hora INT NOT NULL,
  PRIMARY KEY (perfil_id, semana_ref, dia_semana, slot_hora)
);

-- Habilitar Row Level Security (leitura pública)
ALTER TABLE agenda_perfis ENABLE ROW LEVEL SECURITY;
ALTER TABLE agenda_disponibilidade ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON agenda_perfis FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "allow_all" ON agenda_disponibilidade FOR ALL USING (true) WITH CHECK (true);
Encontre em: Supabase Dashboard → Settings → API → Project URL
Encontre em: Supabase Dashboard → Settings → API → anon / public
